# OpenUni Vision

## Mission

OpenUni aims to become the open-source platform that helps university students access information, services, and academic resources through AI.

Instead of searching through dozens of webpages, PDFs, regulations, and FAQs, students should be able to ask a question and receive an accurate answer backed by official university sources.

---

## Long-term vision

OpenUni is not just a RAG chatbot.

It is an extensible platform for universities.

The chatbot is only the first feature.

Over time, OpenUni could become a complete AI-powered university assistant.

---

## Core principles

- Open source
- Official sources only
- Transparent citations
- Privacy-first
- Modular architecture
- Multi-university support

---

---

## Scalability

OpenUni should support multiple universities.

Adding a new university should require only a configuration folder.

Example:

```text
universities/
    unimore/
    unibo/
    polimi/
    unitn/
```

No code changes should be required.
